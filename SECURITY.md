# Security Policy

Use GitHub private vulnerability reporting for `Ezio2000/jharness-python` when a
Python runtime, provider, toolkit, packaging, or transport issue could cause unsafe
execution or data exposure. Do not publish credentials, user data, or exploit details
in a public issue.

Specification-level ambiguities belong in `Ezio2000/jharness`. The latest published
Python minor line receives security fixes; pre-releases may be superseded without a
backport. All three Python distributions receive the same coordinated security version.
