# Python Architecture Decisions

Language-neutral runtime decisions are maintained by the pinned
[JHarness specification](https://github.com/Ezio2000/jharness/tree/v0.1.0/docs/decisions).

This repository records only Python packaging and implementation decisions:

| ADR | Decision |
| --- | --- |
| [0009](0009-product-distributions.md) | Publish exactly three Python product distributions. |

Changing a portable behavior starts in the specification repository. Python-only
decisions update implementation, tests, examples, documentation, and package metadata
together.
